Privacy And Policy - Mobile Notes

Effective Date: April 2026

1. Introduction

Mobile Notes is designed to securely create, store, manage, and back up personal notes. We are committed to protecting user privacy and ensuring secure handling of all user data.

2. Information We Collect

The application may collect limited user information necessary for functionality:

  • Email address during Firebase Authentication sign-in
  • Google account authorization for backup access
  • User-created notes stored locally
  • Backup metadata required for restore process

3. Local Data Storage

All notes are stored locally on the user's device using Android Room Database. This enables offline access without requiring internet connectivity.

4. Data Encryption

Local notes are encrypted using Android Keystore system-generated secure keys. Encryption protects note content stored on the device.

Backup files are converted into encrypted JSON format before upload.

5. Google Drive Backup

Google Drive access is requested only to:

  • Create encrypted note backups
  • Restore encrypted note backups
  • Maintain user-owned backup file

The application does not access unrelated Google Drive files.

6. Firebase Authentication

Firebase Authentication is used for secure account sign-in and identity verification. Firebase may process:

  • Email address
  • User authentication token

7. Account Management

Users can:

  • Sign in securely
  • Switch Google account
  • Sign out
  • Permanently delete account

8. Permanent Account Deletion

Users may permanently delete their account from inside the application. Account deletion removes authentication access and related backup association.

9. Delete Notes

Users can permanently delete notes individually or remove all stored notes. Deleted notes are removed from local storage.

10. Permissions Used

Mobile Notes only requests required permissions:

  • Internet permission
  • Network state access
  • Google account authorization for backup

No unnecessary permissions are requested.

11. Third-Party Services

The application uses:

  • Firebase Authentication
  • Google Drive API

These services follow Google's privacy and security standards.

12. Data Sharing

We do not sell, trade, rent, or share user personal data with third parties.

13. Data Security

Industry-standard encryption and Android platform security features are used to protect stored data.

14. Policy Updates

This Privacy Policy may be updated when application features are modified. Updated versions will be published on this page.
